# Creating Your First Application

If you don't want to deal with other complicated steps and want to start developing an entire project directly with ofa.js, you can use the official online build and preview tool provided by ofa.js to create your first local application in the browser with one click.

This tool is hosted on a static server, and all operations are completed locally in the browser without connecting to a dynamic server, so please use it with confidence.

## Launching the Program

Click [OFA Studio](https://core.noneos.com?redirect=studio) to open the ofa.js build tool. This tool uses the browser's local file system directly, without installation or configuration. Follow these steps:

1. Visit the website and wait for the program to load
2. Click the "Create New Project" button
3. Select "ofa.js Application" type, enter a project name, then select a local directory
4. Click "Allow" on all permission confirmation dialogs that appear, then click "Confirm Create"
5. Wait a few seconds, and the project will be initialized in the selected directory

After the project initialization is complete, you can use editors like [VSCode](https://code.visualstudio.com/) to open the directory and start writing application code.

## Key Points

- **Zero Configuration Development**: No need to install dependencies or configure scaffolding
- **Local Operations**: All operations are completed locally in the browser, no dynamic server needed
- **Quick Start**: Project initialization completes in just a few seconds
- **Editor Friendly**: Supports mainstream editors like VSCode
