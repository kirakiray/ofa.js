# 创建第一个应用

如果你不想折腾其他繁琐步骤，想直接使用 ofa.js 开发整个项目，可以借助 ofa.js 官方提供的在线构建与预览工具，在浏览器里一键创建你的第一个本地应用。

该工具托管于静态服务器，所有操作均在浏览器本地完成，无需连接动态服务器，请放心使用。

## 启动程序

点击 [OFA Studio](https://core.noneos.com?redirect=studio) 打开 ofa.js 构建工具。该工具直接使用浏览器的本地文件系统，无需安装或配置。按照以下步骤操作：

1. 访问网站并等待程序加载
2. 点击 "创建新项目" 按钮
3. 选择 "ofa.js 应用" 类型，输入项目名称，然后选择本地目录
4. 在弹出的权限确认对话框中全部点击允许，然后点击 "确认创建"
5. 等待几秒后，项目将在所选目录中初始化完成

项目初始化完成后，您可使用 [VSCode](https://code.visualstudio.com/) 等编辑器打开该目录，开始编写应用代码。

## 关键要点

- **零配置开发**：无需安装依赖或配置脚手架
- **本地操作**：所有操作在浏览器本地完成，无需动态服务器
- **快速启动**：几秒钟即可完成项目初始化
- **编辑器友好**：支持 VSCode 等主流编辑器
