API-Dokumentation

ofa.js unterstützt nicht nur die Vorlagensyntax, sondern bietet auch eine umfangreiche API, um direkt mit dem DOM zu arbeiten. Wie jQuery können Sie mit der prägnanten ofa.js-Syntax das umständliche native JavaScript ersetzen und DOM-Operationen mühelos durchführen.

In den Lebenszyklusfunktionen und benutzerdefinierten Methoden von Komponenten oder Seitenmodulen können Sie jederzeit die von ofa.js bereitgestellten APIs aufrufen, um DOM-Elemente flexibel und effizient zu steuern.

Instanzbezogene

$(selector)

Kernfunktion in ofa.js zum Abrufen und Manipulieren von DOM-Elementinstanzen

 all

Ruft alle untergeordneten Elementinstanzen des aktuellen Elements ab

 shadow

Ruft das Shadow-Root des aktuellen Elements ab

 prev

Ruft das vorherige Geschwisterelement des aktuellen Elements ab

 prevs

Ruft alle vorherigen Geschwisterelemente des aktuellen Elements ab

 next

Ruft das nächste Geschwisterelement des aktuellen Elements ab

 nexts

Ruft alle nachfolgenden Geschwisterelemente des aktuellen Elements ab

 siblings

Ruft alle Geschwisterelemente des aktuellen Elements ab

 parent

Ruft das übergeordnete Element des aktuellen Elements ab

 parents

Ruft alle Vorfahrenelemente des aktuellen Elements ab

 clone

Kopiert das aktuelle Element und seine untergeordneten Elemente

 ele

Ruft die untergeordnete Elementinstanz des aktuellen Elements ab

 root

Ruft die Wurzelinstanz ab, zu der das aktuelle Element gehört

 Untergeordnete Elemente

Ruft alle untergeordneten Elemente des aktuellen Elements ab

 host

Ruft das Host-Element der Web-Komponente ab, zu der das aktuelle Element gehört

 app

Ruft die o-app-Komponenteninstanz ab, zu der das aktuelle Element gehört

Knotenoperationen

push

Fügt untergeordnete Elemente zum aktuellen Element hinzu

 pop

Entfernt das letzte untergeordnete Element des aktuellen Elements

 shift

Entfernt das erste untergeordnete Element des aktuellen Elements

 unshift

Fügt das erste untergeordnete Element zum aktuellen Element hinzu

 splice

Fügt untergeordnete Elemente hinzu oder entfernt sie

 before

Fügt untergeordnete Elemente vor dem aktuellen Element hinzu

 after

Fügt untergeordnete Elemente nach dem aktuellen Element hinzu

 remove

Entfernt das aktuelle Element

 wrap

Umschließt das aktuelle Element mit einem angegebenen Element

 unwrap

Entfernt die Umschließung des übergeordneten Elements des aktuellen Elements

Attributoperationen

text

Ruft den Textinhalt des Elements ab oder legt ihn fest

 html

Ruft den HTML-Inhalt des Elements ab oder legt ihn fest

 attr

Ruft ein Attribut des Elements ab oder legt es fest

 css

Ruft den Stil des Elements ab oder legt ihn fest

 style

Ruft das style-Objekt des Elements ab

 classList

Ruft die Klassenliste des Elements ab

 data

Ruft das data-Attribut des Elements ab oder legt es fest

Ereignisbezogene

on

Fügt einen Ereignislistener zum Element hinzu

 one

Fügt einen Ereignislistener hinzu, der nur einmal ausgeführt wird

 emit

 off

Entfernt einen Ereignislistener vom Element
