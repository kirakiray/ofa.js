API-Dokumentation

ofa.js unterstützt nicht nur Template-Syntax, sondern bietet auch umfangreiche APIs zur direkten DOM-Manipulation. Ähnlich wie jQuery kannst du mit der prägnanten ofa.js-Syntax das ausführliche native JavaScript ersetzen und mühelos DOM-Operationen durchführen.

In den Lebenszyklusfunktionen und benutzerdefinierten Methoden von Komponenten oder Seitenmodulen kannst du jederzeit die von ofa.js bereitgestellten APIs aufrufen, um DOM-Elemente flexibel und effizient zu steuern.

Instanz bezogen

$(selector)

Die Kernfunktion in ofa.js, die zum Abrufen und Bearbeiten von DOM-Element-Instanzen verwendet wird

 all

Ruft alle untergeordneten Elementinstanzen des aktuellen Elements ab

 shadow

Ruft den Shadow Root des aktuellen Elements ab

 prev

Ruft das vorherige Geschwisterelement des aktuellen Elements ab

 prevs

Ruft alle vorherigen Geschwisterelemente des aktuellen Elements ab

 next

Ruft das nächste Geschwisterelement des aktuellen Elements ab

 nexts

Ruft alle nächsten Geschwisterelemente des aktuellen Elements ab

 siblings

Ruft alle Geschwisterelemente des aktuellen Elements ab

 parent

Ruft das übergeordnete Element des aktuellen Elements ab

 parents

Ruft alle übergeordneten Elemente des aktuellen Elements ab

 clone

Kopiert das aktuelle Element und seine untergeordneten Elemente

 ele

Ruft die untergeordnete Elementinstanz des aktuellen Elements ab

 root

Ruft die Stamm-Instanz ab, zu der das aktuelle Element gehört

 Unterelemente

Ruft alle untergeordneten Elemente des aktuellen Elements ab

 host

Ruft das Host-Element des Web Components ab, zu dem das aktuelle Element gehört

 app

Ruft die o-app-Komponenteninstanz ab, zu der das aktuelle Element gehört

Knotenoperationen

push

Fügt untergeordnete Elemente zum aktuellen Element hinzu

 pop

Entfernt das letzte untergeordnete Element des aktuellen Elements

 shift

Entfernt das erste untergeordnete Element des aktuellen Elements

 unshift

Fügt das erste untergeordnete Element zum aktuellen Element hinzu

 splice

Fügt untergeordnete Elemente zum aktuellen Element hinzu oder entfernt sie

 before

Fügt vor dem aktuellen Element untergeordnete Elemente hinzu

 after

Fügt nach dem aktuellen Element untergeordnete Elemente hinzu

 remove

Entfernt das aktuelle Element

 wrap

Umgibt das aktuelle Element mit einem angegebenen Element

 unwrap

Entfernt die übergeordnete Elementumgebung des aktuellen Elements

Attributoperationen

text

Ruft den Textinhalt des Elements ab oder legt ihn fest

 html

Ruft den HTML-Inhalt des Elements ab oder legt ihn fest

 attr

Ruft Attribute des Elements ab oder legt sie fest

 css

Ruft den CSS-Stil des Elements ab oder legt ihn fest

 style

Ruft das Style-Objekt des Elements ab

 classList

Ruft die Klassenliste des Elements ab

 data

Ruft die data-Attribute des Elements ab oder legt sie fest

Ereignis bezogen

on

Fügt dem Element einen Ereignis-Listener hinzu

 one

Fügt dem Element einen Ereignis-Listener hinzu, der nur einmal ausgeführt wird

 emit

Löst ein benutzerdefiniertes Ereignis des Elements aus

 off

Entfernt den Ereignis-Listener des Elements

Sonstiges

o-app Komponente

Die Stammkomponente der ofa.js-Anwendung, die zum Verwalten des Anwendungsstatus und des Routings verwendet wird

 o-page Komponente

Die Seitenkomponente von ofa.js, die zum Erstellen von Einzelseiten-Anwendungsseiten verwendet wird

 formData

Ruft Formulardaten ab

 tag

Ruft den Tag-Namen des Elements ab

 index

Ruft den Index des Elements in seinem übergeordneten Element ab

 is

Prüft, ob das Element mit dem angegebenen Selektor übereinstimmt

 refresh

Aktualisiert die aktuelle Komponente oder Seite

 PATH

Ruft den Pfad der aktuellen Seite ab

 extend

Erweitert die Funktionalität von ofa.js

 version

Ruft die Versionsnummer von ofa.js ab

 Instanzdatenmerkmale

Stanz ist das reaktive Datenmerkmal von ofa.js, das die automatische Aktualisierung von Daten unterstützt
