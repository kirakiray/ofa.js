API-Dokumentation

ofa.js unterstützt nicht nur Template-Syntax, sondern bietet auch umfangreiche APIs zur direkten DOM-Manipulation. Ähnlich wie jQuery kannst du mit der prägnanten ofa.js-Syntax das ausführliche native JavaScript ersetzen und mühelos DOM-Operationen durchführen.

In den Lebenszyklusfunktionen und benutzerdefinierten Methoden von Komponenten oder Seitenmodulen kannst du jederzeit die von ofa.js bereitgestellten APIs aufrufen, um DOM-Elemente flexibel und effizient zu steuern.

Alle vorherigen Geschwister-Elemente des aktuellen Elements abrufen

 next

Das nächste Geschwister-Element des aktuellen Elements abrufen

 nexts

Alle nachfolgenden Geschwister-Elemente des aktuellen Elements abrufen

Alle Geschwister-Elemente des aktuellen Elements abrufen

 parent

Das Eltern-Element des aktuellen Elements abrufen

 parents

Alle Vorfahren-Elemente des aktuellen Elements abrufen

Das aktuelle Element samt Kinder-Elementen klonen

Kind-Element-Instanz des aktuellen Elements abrufen

Die Root-Instanz, zu der das aktuelle Element gehört, abrufen

Alle Kind-Elemente des aktuellen Elements abrufen

Das Web-Component-Host-Element abrufen, zu dem das aktuelle Element gehört

 unwrap
