# Einführung

## Was ist ofa.js?

ofa.js ist ein sehr benutzerfreundliches Web-Frontend-Framework zur Entwicklung von Web-Anwendungen, das darauf abzielt, die Entwicklungseffizienz zu verbessern und die Einstiegshürden zu senken.

>Wenn Sie mit HTML, CSS und JavaScript **bereits gut vertraut sind**, dann ist ofa.js eine sehr gute Wahl.

## Warum habe ich ofa.js entwickelt?

Obwohl es bereits Frontend-Frameworks wie React und Vue gibt, haben sie alle eine relativ hohe Einstiegshürde. Entwickler müssen zunächst eine Reihe von Vorwissen erlernen, wie z.B. die Installation von Node.js, die Verwendung von npm zum Installieren von Paketen, das Einrichten von Scaffolding usw.

Diese anfänglichen Schritte sind für die eigentliche Entwicklung keineswegs unerlässlich, sondern werden in der Projektstartphase zu einer „Blackbox", die die Anwendung von Anfang an mit unnötiger Komplexität belastet. Die Entstehung von ofa.js zielt genau darauf ab, diese Schwierigkeiten zu überwinden.

## Zweck der Existenz von ofa.js

### Vereinfachung des Frontend-Entwicklungsprozesses

ofa.js vereinfacht die mühsamen Prozesse der traditionellen Frontend-Entwicklung. Entwickler müssen keine Tools wie Node.js, npm und Scaffolding lernen, um schnell Frontend-Anwendungen aufzubauen. Sie müssen lediglich das offizielle Build-Programm (Webversion) im Browser öffnen, dann ein lokales Verzeichnis auswählen und können sofort mit der Entwicklung beginnen.

### Die Einstiegshürden für Entwickler senken

Viele Menschen möchten ihre eigene App entwickeln oder Entwickler werden, doch angesichts der traditionellen Web-Entwicklung (einschließlich Android, iOS usw.) scheuen sie zurück, weil sie zahlreiche Vorkenntnisse erwerben müssen – etwa die Installation der Entwicklungsumgebung und das Einrichten von Scaffolding –, was viele Anfänger abschreckt.

ofa.js widmet sich der Vereinfachung aller Dinge. Sie benötigen nur einen Chrome-Browser, öffnen die Webseite und können sofort Anwendungen erstellen. Wählen Sie direkt ein lokales Verzeichnis aus und öffnen Sie es, ohne Vorbereitungen vor der Entwicklung. Alle Berechnungen, Daten- und Speichervorgänge werden lokal auf dem Rechner des Benutzers ausgeführt, ohne Abhängigkeit von Cloud-Diensten.

### Die ultimative Lösung für Micro-Frontends

ofa.js ermöglicht es, Web-Front-End-Entwicklung wie Microservices in mehrere unabhängige Module aufzuteilen, die jeweils eigenständig entwickelt und bereitgestellt werden können. Nachdem die traditionellen Grenzen des Web-Front-Ends durchbrochen wurden, wird die Front-End-Technologie schrittweise die Grenzen der Server-Technologie überschreiten.

> Microservices (Microservices) ist ein Softwarearchitekturstil, der große, komplexe Anwendungen in mehrere feinkörnige, unabhängig bereitgestellte und ausgeführte kleine Dienste aufteilt.
