# 创建第一个应用

您可以直接使用 ofa.js 提供的在线构建工具，在浏览器中创建您的第一个本地应用。

## 启动程序

点击 [OFA Studio](https://core.noneos.com/nos-tool/studio/) 打开 ofa.js 构建工具。该工具直接使用浏览器的本地文件系统，无需安装或配置。按照以下步骤操作：

1. 访问网站并等待程序加载
2. 选择 "ofa.js 应用" 类型，输入项目名称，然后选择本地目录
3. 在弹出的权限确认对话框中全部点击允许，然后点击 "确认创建"
4. 等待几秒后，项目将在所选目录中初始化完成

项目初始化完成后，您可使用 VSCode 等编辑器打开该目录，开始编写应用代码。所有操作均通过浏览器在本地完成，没有连接动态服务器，请放心使用。
