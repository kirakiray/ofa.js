# 介绍

## 什么是 ofa.js？

ofa.js 是一款易用的 Web 前端框架，专为构建 Web 应用而设计，旨在提高开发效率，降低开发门槛。

> 如果你对 HTML、CSS 和 JavaScript **已经基本熟悉**，那么 ofa.js 是一个非常好的选择。

## 设计理念

传统应用开发需要应对不同的硬件底层和操作系统，因此通过编译来实现跨平台无差别化。但 Web 前端天生无需面对这些问题，它的优势在于即刻运行——代码修改后能迅速看到效果，这是浏览器与生俱来的优势。

然而，近十年的前端开发沿用了传统应用开发的经验，引入 Node.js 等工具对网页应用进行"编译"，让事情变得异常复杂，使 Web 失去了即刻运行的核心优势。沿着编译这条路走下去，最终都难以避免"巨石应用"的困境。

传统的 Web 前端应用天生就是微前端架构，但为了解决模块化和工程化问题，却丧失了"无需编译"这一巨大优势。

ofa.js 的出发点很简单：Web 开发在解决模块化和工程化的同时，应该保留快速开发的易用性——不依赖编译流程，这才是正确的发展方向。

## 核心优势

### 零门槛，开箱即用

无需搭建开发环境、安装依赖或配置脚手架，通过浏览器打开官方构建程序，选择本地目录即可开始开发。所有计算、数据和存储操作均在本地执行，无需依赖云端服务。

### AI 友好，易于验证

没有编译黑箱，AI 产出的代码可以快速部署和自验证；减少中间层，不走编译流程，让代码更容易定位问题和修复。

### 微前端原生支持

ofa.js 让 Web 前端开发可以像微服务一样，细分为多个独立模块，每个模块均可独立开发、独立部署。当传统 Web 前端的边界被打破后，前端技术将逐步突破服务器技术的边界。

> 微服务（Microservices）是一种软件架构风格，它将大型、复杂的应用拆分为多个细粒度、独立部署和运行的小型服务。
