# 介绍

## 什么是 ofa.js？

ofa.js 是一个 Web 前端框架，用于构建 Web 应用，旨在提高开发效率，降低开发门槛。

## 为什么我要开发 ofa.js？

虽然已有 React、Vue 等前端框架，但它们都存在较高的上手门槛。开发者需要先学习一系列前置知识，例如安装 Node.js、使用 npm 安装包、搭建脚手架等。

这些初始步骤对实际开发并非不可或缺，反而成为了项目启动阶段的“黑箱”，让应用从一开始就背负了不必要的复杂度。ofa.js 的诞生，正是为了打破这一困境。

## ofa.js 存在的目的

### 简化前端开发流程

ofa.js 简化了传统前端开发的繁琐流程。开发者无需学习 Node.js、npm 和脚手架等工具，即可快速搭建前端应用。只需通过浏览器打开官方提供的构建程序（网页版），然后选择本地目录，便可开始开发。

### 降低开发门槛

许多人希望为自己开发一款应用，或成为一名开发者，但面对传统Web（包括安卓、iOS等）开发路线时，却因缺乏编程基础而望而却步。当今任何一条开发路线都需要学习大量预备知识，这让许多初学者知难而退。

ofa.js 致力于让一切变得简单。只需一个 Chrome 浏览器，打开网页即可构建应用，直接选择并打开本地目录，省去开发前的准备工作。所有计算、数据和存储操作均在用户本地执行，无需依赖云端服务。

### 成为微前端的终极解决方案

ofa.js 让 Web 前端开发可以像微服务一样，细分为多个独立模块，每个模块均可独立开发、独立部署。当传统 Web 前端的边界被打破后，前端技术将逐步突破服务器技术的边界。
