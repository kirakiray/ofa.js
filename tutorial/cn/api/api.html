API 文档

ofa.js 不仅支持模板语法，还提供丰富的 API 来直接操作 DOM。就像 jQuery 一样，你可以用简洁的 ofa.js 语法替代冗长的原生 JavaScript，轻松完成 DOM 操作。

在组件或页面模块的生命周期函数及自定义方法中，你都可以随时调用 ofa.js 提供的 API，灵活高效地操控 DOM 元素。

实例相关

$(selector)

ofa.js 中的核心函数，用于获取和操作 DOM 元素实例

all

获取当前元素的所有子元素实例

 shadow

获取当前元素的 shadow root

 prev

获取当前元素的前一个兄弟元素

 prevs

获取当前元素之前的所有兄弟元素

 next

获取当前元素的下一个兄弟元素

 nexts

获取当前元素之后的所有兄弟元素

 siblings

获取当前元素的所有兄弟元素

 parent

获取当前元素的父元素

 parents

获取当前元素的所有祖先元素

 clone

复制当前元素及其子元素

 ele

获取当前元素的子元素实例

 root

获取当前元素所属的根实例

 子元素

获取当前元素的所有子元素

 host

获取当前元素所属的 web component 宿主元素

 app

获取当前元素所属的 o-app 组件实例

节点操作

push

添加当前元素的子元素

 pop

删除当前元素的最后一个子元素

 shift

删除当前元素的第一个子元素

 unshift

添加当前元素的第一个子元素

 splice

添加或删除当前元素的子元素

 before

在当前元素之前添加子元素

 after

在当前元素之后添加子元素

 remove

移除当前元素

 wrap

将当前元素包裹在指定元素中

 unwrap

移除当前元素的父元素包裹

属性操作

text

获取或设置元素的文本内容

 html

获取或设置元素的 HTML 内容

 attr

获取或设置元素的属性

 css

获取或设置元素的样式

 style

获取元素的 style 对象

 classList

获取元素的类名列表

 data

获取或设置元素的 data 属性

事件相关

on

为元素添加事件监听器

 one

为元素添加只执行一次的事件监听器

 emit

触发元素的自定义事件

 off

移除元素的事件监听器

 o-page组件

ofa.js 的页面组件，用于创建单页应用页面

 formData

 tag

获取元素的标签名

 index

获取元素在其父元素中的索引

 is

判断元素是否匹配指定选择器

 refresh

刷新当前组件或页面

 PATH

获取当前页面的路径

 extend

扩展 ofa.js 的功能

 version

获取 ofa.js 的版本号
