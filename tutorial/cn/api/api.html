API 文档

ofa.js 不仅支持模板语法，还提供丰富的 API 来直接操作 DOM。就像 jQuery 一样，你可以用简洁的 ofa.js 语法替代冗长的原生 JavaScript，轻松完成 DOM 操作。

在组件或页面模块的生命周期函数及自定义方法中，你都可以随时调用 ofa.js 提供的 API，灵活高效地操控 DOM 元素。

实例相关

$(selector)

$方法是 ofa.js 中的核心函数，用于获取和操作 DOM 元素实例

all

all方法用于获取当前元素的所有子元素实例

 ele

ele方法用于获取当前元素的子元素实例

节点操作

push

push方法用于添加当前元素的子元素

 pop

pop方法用于删除当前元素的最后一个子元素

 shift

shift方法用于删除当前元素的第一个子元素

 unshift

unshift方法用于添加当前元素的第一个子元素

 splice

splice方法用于添加或删除当前元素的子元素

 before

before方法用于在当前元素之前添加子元素
