I’m too old to learn, don’t make me read docs—can I still use it?

You can reuse components built with ofa.js without learning it first.

After the open-source author finishes the code, anyone can copy a few lines from the demo and instantly drop the feature into their own project.

I need to bolt a quick feature onto an ancient project

Just drop a script tag into your existing project and ofa.js is ready to go.

After that, every kind of component is at your fingertips.

I’m a total newbie—Node.js and npm already scared me off

ofa.js is beginner-friendly—no need to learn Node.js, npm, or any tooling.

All you need is a browser and any static server to start coding.

If you use Chrome, open the online tool OFA Studio and you can skip the static server entirely—just start coding.

Tutorial: Create Your First App
Building a massive shopping-mall system—can you handle it?

ofa.js is perfect for large projects thanks to its micro-frontend architecture.

You can slice the mall into tiny requirements, develop each component in its own folder, and preview it instantly via a static server.

Finally assemble them into a gigantic system. ofa.js’s micro-frontend model is the cure for monolithic apps.

See the real-world example NoneOS—an operating system built with ofa.js, massive in scale.

I’m a backend dev—just want to use server-side templating. Is ofa.js okay?

ofa.js SSR isn’t tied to Node.js; as long as you follow the SCSR rules you can use any backend language for templating.

jQuery's Spiritual Successor

Supports both template syntax and direct node manipulation, ensuring both flexibility and efficiency

API design is similar to jQuery, but more intuitive

      
// jQuery Code
$("#target").html("some html code"); // Set html
$("#target").text("set text"); // Set text
var ele_text = $("#target").text(); // Get text
var parents = $("#target").parents(); // Get an array of all ancestors
var child = $("#target").children()[0]; // Get first child element
      
    
      
// ofa.js Code
$("#target").html = "some html code";  // Set html
$("#target").text = "set text"; // Set text
var ele_text = $("#target").text; // Get text
var parents = $("#target").parents; // Get an array of all ancestors
var child = $("#target")[0]; // Get first child element
