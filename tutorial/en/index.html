I'm too tired to learn more—can I still use it without reading your docs?

You can reuse components built with ofa.js without learning it first.

After authors publish open-source components, anyone can copy a few lines from the demo and instantly drop the feature into their own project.

I need to bolt a quick feature onto an old legacy project

Just drop in ofa.js with a single script tag and start using it immediately.

Once included, countless components are at your fingertips.

I'm a beginner scared off by Node.js and npm

ofa.js is beginner-friendly—no need for Node.js, npm or any toolchain.

All you need is a browser and any static server to start developing.

If you use Chrome, open the online tool OFA Studio and you can skip the static server entirely.

Tutorial: Create your first app
Building a huge shopping-mall system—can you handle it?

ofa.js excels at large projects thanks to its micro-frontends architecture.

Split the mall into tiny requirements, develop each component in its own folder, and preview it instantly via any static server.

Assemble them piece by piece into a massive system. ofa.js micro-frontends are the perfect cure for monolithic apps.

See the real-world example NoneOS—an entire operating system built with ofa.js, proving its scale.

I'm a backend dev—can I stick to server-side templating with ofa.js?

ofa.js SSR is not tied to Node.js; follow the SCSR rules and you can use any backend language for templating.

jQuery's Spiritual Successor

Supports both template syntax and direct node manipulation, ensuring both flexibility and efficiency

API design is similar to jQuery, but more intuitive

      
// jQuery Code
$("#target").html("some html code"); // Set html
$("#target").text("set text"); // Set text
var ele_text = $("#target").text(); // Get text
var parents = $("#target").parents(); // Get an array of all ancestors
var child = $("#target").children()[0]; // Get first child element
      
    
      
// ofa.js Code
$("#target").html = "some html code";  // Set html
$("#target").text = "set text"; // Set text
var ele_text = $("#target").text; // Get text
var parents = $("#target").parents; // Get an array of all ancestors
var child = $("#target")[0]; // Get first child element
