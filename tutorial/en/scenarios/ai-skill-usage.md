# Using ofa.js AI Skills in Projects

Since ofa.js has not yet become a widely recognized framework, mainstream AI models do not yet have the ability to directly use ofa.js. To address this, we have carefully prepared a dedicated skill package to help AI learn and reference how to use ofa.js.

## Usage

In AI tools that support skills, select the corresponding skill address based on the AI model language you are using:

**Note:** Since the documentation for ofa.js was originally written in Chinese, the Chinese version of the skill package is more complete and accurate. If you encounter problems while using the English version, it is recommended to try the Chinese version.

### Chinese Model

If you are using a Chinese AI model, please use the following address:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/skills/ofajs-docs.zip
```

Common Chinese AI models include: DeepSeek, Kimi, GLM, MiniMax, Qwen, etc.

### English Model

If you are using an English AI model, please use the following address:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/skills/ofajs-docs-en.zip
```

With this skill package, AI can efficiently learn and develop ofa.js components or page modules, helping developers more conveniently use AI tools for ofa.js project development and improve development efficiency.
