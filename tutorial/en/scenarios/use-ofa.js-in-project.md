# AI Prompts for Using ofa.js in Projects

Since ofa.js is not yet a widely known framework, mainstream AI models do not have the ability to use ofa.js directly. To address this, we have carefully prepared dedicated prompts to help AI learn and reference how to use ofa.js.

We provide two versions of prompts:

## Streamlined Prompt Words

This is a condensed and optimized version designed to minimize the consumption of prefix tokens, suitable for most scenarios:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/tiny/start.md
```

Using this prompt, AI can efficiently develop components or page modules for ofa.js.

## Complete Version Prompt

If the AI model you are using is relatively less intelligent, you can try using the full, unconcentrated version of the prompt. Although it will occupy more tokens during initialization, it may yield better results:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/origin/start.md
```

By providing these prompts, we hope to help developers more conveniently leverage AI tools for ofa.js project development and improve development efficiency.
