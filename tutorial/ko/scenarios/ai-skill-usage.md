# 프로젝트에서 ofa.js의 AI 스킬 사용하기

ofa.js는 아직 널리 알려진 프레임워크가 아니기 때문에 주류 AI 모델은 ofa.js를 직접 사용할 수 있는 능력을 갖추지 못했습니다. 이를 위해 우리는 AI가 ofa.js의 사용법을 학습하고 참고할 수 있도록 전용 스킬 팩을 정성껏 준비했습니다.

## 사용 방법

지원 스킬을 갖춘 AI 도구에서 사용 중인 AI 모델 언어에 따라 해당 스킬 주소를 선택하세요:

**주의:** ofa.js 문서는 원래 중국어로 작성되었기 때문에 중국어 버전의 스킬팩 내용이 더 완전하고 정확합니다. 영어 버전 사용 시 문제가 발생하면 중국어 버전 사용을 권장합니다.

### 중국어 모델

중국어 AI 모델을 사용 중이라면 다음 주소를 이용하세요:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/skills/ofajs-docs.zip
```

일반적인 중국 AI 모델에는 DeepSeek, Kimi, GLM, MiniMax, Qwen 등이 있습니다.

### 영어 모델

만약 영어 AI 모델을 사용 중이시라면, 다음 주소를 이용해 주세요:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/skills/ofajs-docs-en.zip
```

이 스킬 팩을 통해 AI는 ofa.js의 컴포넌트나 페이지 모듈을 효율적으로 학습하고 개발할 수 있으며, 개발자가 AI 도구를 더 편리하게 활용하여 ofa.js 프로젝트를 개발할 수 있도록 도와 개발 효율성을 높여줍니다.

## 더 많은 도움말 얻기

AI 스킬로 문제가 해결되지 않았다면, 토론 게시판에 질문을 남겨 주세요:

https://github.com/ofajs/discussion/discussions

최대한 신속하게 답변 드리겠습니다.
