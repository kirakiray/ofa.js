# 프로젝트에서 ofa.js의 AI 프롬프트 사용하기

ofa.js는 현재 널리 알려진 프레임워크가 아니기 때문에, 주류 AI 모델이 직접 ofa.js를 사용할 수 있는 능력을 갖추고 있지 않습니다. 이에 따라, 우리는 AI가 ofa.js의 사용 방법을 학습하고 참조할 수 있도록 전용 프롬프트를 정성껏 준비했습니다.

우리는 두 가지 버전의 프롬프트를 제공합니다:

## 간소화 프롬프트

이것은 압축 및 최적화된 버전으로, 프리픽스 토큰 입력 소비를 최소화하도록 설계되었으며 대부분의 시나리오에 적합합니다:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/tiny/start.md
```

이 프롬프트를 사용하면 AI는 ofa.js의 컴포넌트나 페이지 모듈을 효율적으로 개발할 수 있습니다.

## 전체 버전 프롬프트

사용하는 AI 모델이 상대적으로 지능이 부족한 경우, 농축되지 않은 완전판 프롬프트를 사용해 볼 수 있습니다. 초기화 시 더 많은 토큰을 차지하지만 더 나은 효과를 얻을 수 있습니다:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/origin/start.md
```

이러한 프롬프트를 제공함으로써, 우리는 개발자들이 AI 도구를 더 편리하게 활용하여 ofa.js 프로젝트를 개발하고 개발 효율성을 높일 수 있기를 바랍니다.
