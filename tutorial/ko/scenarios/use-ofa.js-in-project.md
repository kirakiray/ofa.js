# 프로젝트에서 ofa.js의 AI 프롬프트 사용하기

현재 ofa.js가 널리 알려진 프레임워크로 자리 잡지 않았기 때문에, 주요 AI 모델은 아직 ofa.js를 직접 활용하는 능력이 부족합니다. 이에 저희는 AI가 ofa.js의 사용법을 학습하고 참고할 수 있도록 전용 프롬프트를 정성껏 준비했습니다.

우리는 두 가지 버전의 프롬프트를 제공합니다

## 간소화된 프롬프트

이것은 전처리 토큰 입력 소비를 최소화하도록 농축 최적화된 버전으로, 대부분의 시나리오에 적합합니다:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/tiny/start.md
```

이 프롬프트를 사용하면 AI가 ofa.js의 컴포넌트나 페이지 모듈을 효율적으로 개발할 수 있습니다.

## 전체 버전 프롬프트

사용 중인 AI 모델이 상대적으로 덜 똑똑한 경우, 농축되지 않은 전체 프롬프트를 사용해 보세요. 초기화 시 더 많은 토큰을 차지하지만 더 나은 효과를 얻을 수 있습니다:

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/origin/start.md
```

이러한 프롬프트를 제공함으로써, 우리는 개발자가 AI 도구를 활용하여 ofa.js 프로젝트 개발을 더욱 편리하게 진행하고, 개발 효율을 향상시키는 데 도움을 주고자 합니다.
