튜토리얼: 첫 앱 만들기
거대한 쇼핑몰 시스템을 개발하려는데, 너희가 감당할 수 있어?

ofa.js는 마이크로 프론트엔드 아키텍처 패턴을 채택하고 있어 대형 프로젝트에 매우 적합합니다.

쇼핑몰 시스템을 충분히 세분화된 요구사항으로 나누어, 컴포넌트 단위로 개발한 뒤 독립적인 폴더에 배치하면 정적 서버만으로도 바로 컴포넌트 효과를 미리 볼 수 있습니다.

마지막으로 하나씩 조립해 거대한 시스템을 완성합니다. ofa.js의 마이크로 프론트엔드 방식은 바로 거대 애플리케이션을 해결하는 묘약입니다.

jQuery의 정신적 계승자

템플릿 구문을 지원하면서도 직접 노드를 조작할 수 있어 유연성과 효율성을 모두 잡았습니다

API 설계는 jQuery와 유사하지만 더 직관적입니다

      
// jQuery 코드
$("#target").html("some html code"); // html 설정
$("#target").text("set text"); // text 설정
var ele_text = $("#target").text(); // text 가져오기
var parents = $("#target").parents(); // 모든 부모 계층 배열 가져오기
var child = $("#target").children()[0]; // 첫 번째 자식 요소 가져오기
      
    
      
// ofa.js 코드
$("#target").html = "some html code";  // html 설정
$("#target").text = "set text"; // text 설정
var ele_text = $("#target").text; // text 가져오기
var parents = $("#target").parents; // 모든 부모 계층 배열 가져오기
var child = $("#target")[0]; // 첫 번째 자식 요소 가져오기
