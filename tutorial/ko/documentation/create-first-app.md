# 첫 번째 앱 만들기

번거로운 다른 단계를 거치고 싶지 않고, ofa.js를 사용하여 전체 프로젝트를 바로 개발하고 싶다면, ofa.js 공식에서 제공하는 온라인 빌드 및 미리보기 도구를 활용하여 브라우저에서 한 번에 첫 번째 로컬 애플리케이션을 만들 수 있습니다.

이 도구는 정적 서버에 호스팅되며, 모든 작업은 브라우저 로컬에서 수행되므로 동적 서버에 연결할 필요가 없습니다. 안심하고 사용하세요.

## 프로그램 시작

[OFA Studio](https://core.noneos.com?redirect=studio)를 클릭하여 ofa.js 빌드 도구를 엽니다. 이 도구는 브라우저의 로컬 파일 시스템을 직접 사용하며, 설치나 설정이 필요 없습니다. 다음 단계에 따라 작업하세요:

1. 웹사이트에 접속하여 프로그램이 로드될 때까지 기다리세요.
2. "새 프로젝트 만들기" 버튼을 클릭하세요.
3. "ofa.js 앱" 유형을 선택하고, 프로젝트 이름을 입력한 후 로컬 디렉토리를 선택하세요.
4. 팝업된 권한 확인 대화상자에서 모두 허용을 클릭한 후, "만들기 확인"을 클릭하세요.
5. 몇 초 후 선택한 디렉토리에 프로젝트가 초기화됩니다.

프로젝트 초기화가 완료되면 [VSCode](https://code.visualstudio.com/) 등의 편집기로 해당 디렉토리를 열어 애플리케이션 코드 작성을 시작할 수 있습니다.
