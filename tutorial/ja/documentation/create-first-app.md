# 最初のアプリケーションを作成する

もし面倒な手順を踏みたくなく、ofa.js を使ってプロジェクト全体を開発したい場合は、ofa.js 公式が提供するオンライン構築・プレビューツールを活用すれば、ブラウザでワンクリックで最初のローカルアプリケーションを作成できます。

このツールは静的サーバーでホストされており、すべての操作はブラウザのローカルで完了します。動的サーバーに接続する必要はありませんので、安心してお使いください。

## 起動プログラム

[OFA Studio](https://core.noneos.com?redirect=studio) をクリックして、ofa.js ビルドツールを開きます。このツールはブラウザのローカルファイルシステムを直接使用するため、インストールや設定は不要です。以下の手順に従って操作してください：

1. ウェブサイトにアクセスし、プログラムの読み込みを待つ
2. 「新規プロジェクト作成」ボタンをクリックする
3. 「ofa.js アプリケーション」タイプを選択し、プロジェクト名を入力して、ローカルディレクトリを選択する
4. 表示された権限確認ダイアログで全て許可をクリックし、その後「作成を確認」をクリックする
5. 数秒待つと、選択したディレクトリにプロジェクトが初期化完了する

プロジェクトの初期化が完了したら、[VSCode](https://code.visualstudio.com/) などのエディタでこのディレクトリを開き、アプリケーションコードの記述を開始できます。
