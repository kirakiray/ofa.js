# 最初のアプリケーションを作成する

もし他の面倒な手順を試したくなく、ofa.jsを使ってプロジェクト全体を直接開発したい場合は、ofa.js公式が提供するオンライン構築とプレビューツールを利用して、ブラウザ内でワンクリックで最初のローカルアプリを作成できます。

このツールは静的サーバーにホストされており、すべての操作はブラウザのローカルで行われます。動的サーバーへの接続は不要ですので、安心してご利用ください。

## プログラムの起動

[OFA Studio](https://core.noneos.com?redirect=studio) をクリックして、ofa.js ビルドツールを開きます。このツールはブラウザのローカルファイルシステムを直接使用し、インストールや設定は不要です。以下の手順に従って操作してください：

1. ウェブサイトにアクセスし、プログラムの読み込みを待つ  
2. 「新規プロジェクトを作成」ボタンをクリック  
3. 「ofa.js アプリケーション」タイプを選択し、プロジェクト名を入力してからローカルディレクトリを選択  
4. 表示される権限確認ダイアログですべて「許可」をクリックし、「作成を確認」をクリック  
5. 数秒待つと、選択したディレクトリにプロジェクトが初期化される

プロジェクトの初期化が完了したら、[VSCode](https://code.visualstudio.com/) などのエディタでこのディレクトリを開き、アプリケーションコードの記述を開始できます。
