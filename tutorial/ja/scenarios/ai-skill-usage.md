# プロジェクトでofa.jsのAIスキルを活用する

由于 ofa.js 目前尚未成为广泛知名的框架，主流 AI 模型尚不具备直接使用 ofa.js 的能力。为此，我们精心准备了专用技能包，帮助 AI 学习和查阅 ofa.js 的使用方法。

## 使用方法

スキル対応のAIツールでは、ご使用のAIモデルの言語に応じて対応するスキルアドレスを選択してください：

**注意：** ofa.js のドキュメントは元々中国語で書かれているため、中国語バージョンのスキルパックの内容はより完全で正確です。英語バージョンで問題が発生した場合は、中国語バージョンを試すことをお勧めします。

### 中国語モデル

日本語版AIモデルをご利用の場合は、以下のアドレスをご使用ください：

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/skills/ofajs-docs.zip
```

一般的な中国のAIモデルには、DeepSeek、Kimi、GLM、MiniMax、Qwenなどがあります。

### 英語モデル

如果您使用的是英文 AI 模型，请使用以下地址：

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/skills/ofajs-docs-en.zip
```

このスキルパックにより、AI は ofa.js のコンポーネントやページモジュールを効率的に学習・開発でき、開発者は AI ツールをより手軽に活用して ofa.js プロジェクトの開発を行い、開発効率を向上させることができます。

## さらにヘルプを見る

AIスキルで問題が解決しなかった場合は、お気軽にディスカッションエリアにご投稿ください：

https://github.com/ofajs/discussion/discussions

できるだけ早くご回答いたします。
