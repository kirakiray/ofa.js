# プロジェクトでofa.jsのAIプロンプトを使用する

ofa.jsは現在、広く知られたフレームワークではないため、主流のAIモデルはofa.jsを直接利用する能力をまだ持っていません。そのため、私たちはAIがofa.jsの使用方法を学び、参照できるよう、専用のプロンプトを丁寧に用意しました。

私たちは2つのバージョンのプロンプトを提供しています：

## 簡易版プロンプト

これは濃縮最適化されたバージョンであり、前置きトークンの入力消費を最小限に抑えることを目的としており、ほとんどのシナリオに適用されます：

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/tiny/start.md
```

このプロンプトを使用することで、AI は ofa.js のコンポーネントやページモジュールを効率的に開発できます。

## 完全版プロンプト

使用しているAIモデルが比較的賢くない場合は、濃縮されていない完全版のプロンプトを試してみてください。初期化時により多くのトークンを消費しますが、より良い効果が得られる可能性があります：

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/origin/start.md
```

これらのプロンプトを提供することで、開発者が AI ツールをより便利に活用して ofa.js プロジェクトの開発を行い、開発効率を向上させることを目指しています。
