# プロジェクトでofa.jsのAIプロンプトを使用する

ofa.jsは現在まだ広く知られたフレームワークではないため、主流のAIモデルはofa.jsを直接利用する能力を持っていません。そのため、私たちはAIがofa.jsの使用方法を学び、参照できるように専用のプロンプトを丁寧に用意しました。

私たちは2つのバージョンのプロンプトを提供しています：

## 簡潔版プロンプト

これは濃縮最適化されたバージョンで、最小限の前置トークン入力消費を目指し、ほとんどのシナリオに適用可能です：

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/tiny/start.md
```

このプロンプトを使用すると、AIはofa.jsのコンポーネントやページモジュールを効率的に開発できます。

## 完全版プロンプト

使用しているAIモデルが比較的インテリジェントでない場合は、未濃縮の完全版プロンプトを試してみてください。初期化時により多くのトークンを消費しますが、より良い結果が得られる可能性があります：

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/origin/start.md
```

これらのプロンプトを提供することで、開発者がAIツールを活用してofa.jsプロジェクトの開発をより便利に行い、開発効率を向上させることを支援したいと考えています。
