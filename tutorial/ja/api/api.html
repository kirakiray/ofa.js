API ドキュメント

ofa.jsはテンプレート構文のサポートだけでなく、DOMを直接操作するための豊富なAPIも提供します。 jQuery のように、長いネイティブJavaScriptの代わりに简洁なofa.js構文を使用して、気軽にDOM操作を完了できます。

コンポーネントやページモジュールのライフサイクル関数やカスタムメソッド内では、ofa.jsが提供するAPIをいつでも呼び出して、DOM要素を柔軟かつ効率的に操作することができます。

インスタンス関連

$(selector)

ofa.js の中核となる関数で、DOM 要素インスタンスの取得と操作に使用

 all

現在の要素のすべての子要素インスタンスを取得

 shadow

現在の要素の shadow root を取得

 prev

現在の要素の前の兄弟要素を取得

 prevs

現在の要素より前のすべての兄弟要素を取得

 next

現在の要素の次の兄弟要素を取得

 nexts

現在の要素より後のすべての兄弟要素を取得

 siblings

現在の要素のすべての兄弟要素を取得

 parent

現在の要素の親要素を取得

 parents

現在の要素のすべての祖先要素を取得

 clone

現在の要素とその子要素を複製

 ele

現在の要素の子要素インスタンスを取得

 root

現在の要素が属するルートインスタンスを取得

 子要素

現在の要素のすべての子要素を取得

 host

現在の要素が所属する web component ホスト要素を取得

 app

現在の要素が所属する o-app コンポーネントインスタンスを取得

ノード操作

push

現在の要素に子要素を追加

 pop

現在の要素の最後の子要素を削除

 shift

現在の要素の最初の子要素を削除

 unshift

現在の要素の最初に子要素を追加

 splice

現在の要素の子要素を追加または削除

 before

現在の要素の前に子要素を追加

 after

現在の要素の後に子要素を追加

 remove

現在の要素を削除

 wrap

現在の要素を指定要素で包む

 unwrap

現在の要素の親要素のラップを解除

属性操作

text

要素のテキストコンテンツを取得または設定

 html

要素の HTML コンテンツを取得または設定

 attr

要素の属性を取得または設定

 css

要素のスタイルを取得または設定

 style

要素の style オブジェクトを取得

 classList

要素のクラス名リストを取得

 data

要素の data 属性を取得または設定

イベント関連

on

要素にイベントリスナーを追加

 one

要素に一度だけ実行されるイベントリスナーを追加

 emit

要素のカスタムイベントをトリガー

 off

要素のイベントリスナーを削除

その他

o-appコンポーネント

ofa.js アプリケーションのルートコンポーネントで、アプリケーション状態とルーティングを管理

 o-pageコンポーネント

ofa.js のページコンポーネントで、シングルページアプリケーションのページを作成

 formData

フォームデータを取得

 tag

要素のタグ名を取得

 index

要素が親要素内でのインデックスを取得

 is

要素が指定セレクタに一致するかを判定

 refresh

現在のコンポーネントまたはページをリフレッシュ

 PATH

現在のページパスを取得

 extend

ofa.js の機能を拡張

 version

ofa.js のバージョン番号を取得
