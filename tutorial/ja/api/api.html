API ドキュメント

ofa.js はテンプレート構文をサポートするだけでなく、DOMを直接操作するための豊富なAPIも提供します。 jQuery と同様に、簡潔なofa.jsの構文で冗長なネイティブJavaScriptを置き換え、DOM操作を簡単に行えます。

コンポーネントやページモジュールのライフサイクル関数やカスタムメソッドの中で、いつでもofa.jsが提供する API、を呼び出してDOM要素を柔軟かつ効率的に操作できます。

インスタンス関連

$(selector)

ofa.jsのコア関数で、DOM要素インスタンスを取得・操作するために使用します

 all

現在の要素のすべての子要素インスタンスを取得します

 shadow

現在の要素のシャドウルートを取得します

 prev

現在の要素の前の兄弟要素を取得します

 prevs

現在の要素より前のすべての兄弟要素を取得します

 next

現在の要素の次の兄弟要素を取得します

 nexts

現在の要素より後のすべての兄弟要素を取得します

 siblings

現在の要素のすべての兄弟要素を取得します

 parent

現在の要素の親要素を取得します

 parents

現在の要素のすべての祖先要素を取得します

 clone

現在の要素とその子要素を複製します

 ele

現在の要素の子要素インスタンスを取得します

 root

現在の要素が属するルートインスタンスを取得します

 子要素

現在の要素のすべての子要素を取得します

 host

現在の要素が属するウェブコンポーネントのホスト要素を取得します

 app

現在の要素が属するo-appコンポーネントインスタンスを取得します

ノード操作

push

現在の要素に子要素を追加します

 pop

現在の要素の最後の子要素を削除します

 shift

現在の要素の最初の子要素を削除します

 unshift

現在の要素の最初の子要素として追加します

 splice

現在の要素の子要素を追加または削除します

 before

現在の要素の前に子要素を追加します

 after

現在の要素の後に子要素を追加します

 remove

現在の要素を削除します

 wrap

現在の要素を指定した要素でラップします

 unwrap

現在の要素の親要素によるラップを解除します

プロパティ操作

text

要素のテキストコンテンツを取得または設定します

 html

要素のHTMLコンテンツを取得または設定します

 attr

要素の属性を取得または設定します

 css

要素のスタイルを取得または設定します

 style

要素のstyleオブジェクトを取得します

 classList

要素のクラス名リストを取得します

 data

要素のdata属性を取得または設定します

イベント関連

on

要素にイベントリスナーを追加します

 one

要素に一度だけ実行されるイベントリスナーを追加します

 emit

要素のカスタムイベントを発火します

 off

要素のイベントリスナーを削除します

その他

o-appコンポーネント

ofa.jsアプリケーションのルートコンポーネントで、アプリケーションの状態とルーティングを管理するために使用します

 o-pageコンポーネント

ofa.jsのページコンポーネントで、シングルページアプリケーションのページを作成するために使用します

 formData

フォームデータを取得します

 tag

要素のタグ名を取得します

 index

要素の親要素内でのインデックスを取得します

 is

要素が指定されたセレクタに一致するかどうかを判定します

 refresh

現在のコンポーネントまたはページをリフレッシュします

 PATH

現在のページのパスを取得します

 extend

ofa.jsの機能を拡張します

 version

ofa.jsのバージョン番号を取得します

 インスタンスデータ機能

Stanzはofa.jsのリアクティブデータ機能で、データの自動更新をサポートします
