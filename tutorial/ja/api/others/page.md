# o-page コンポーネント

`o-page` は ofa.js のコアコンポーネントの1つであり、独立したページまたはページモジュールを表します。以下は `o-page` の主なプロパティとメソッドです：

## src 属性

`src`属性は、ページモジュールの具体的なアドレスを指定するために使用されます。これはページの内容と動作を指定する重要な属性で、アプリケーションに特定ページのコンテンツをどこから読み込むかを伝えます。

```javascript
const page = this;
```

## goto メソッド

`goto` メソッドは、現在のページから別のページへジャンプするために使用されます。`app` の `goto` メソッドと比べて、`page` の `goto` メソッドは**相対アドレス**を使って他のページへナビゲートできます。

```javascript
page.goto("./page2.html");
```

## replace メソッド

`replace` メソッドは、現在のページを別のページに置き換えるために使用されます。これは `app` の `replace` メソッドに似ていますが、ページ内で置き換え操作を行う点が異なります。

```javascript
page.replace("./new-page.html");
```

## backメソッド

`back` メソッドは前のページに戻るために使用されます。これはブラウザの戻る操作と同様に、ユーザーを前のページにナビゲートします。

```javascript
page.back();
```
