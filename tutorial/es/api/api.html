Documentación de la API

ofa.js no solo admite sintaxis de plantilla, sino que también proporciona una rica API para manipular directamente el DOM. Al igual que jQuery, puedes utilizar la sintaxis concisa de ofa.js en lugar del prolijo JavaScript nativo para realizar operaciones DOM fácilmente.

En las funciones del ciclo de vida de componentes o módulos de página y en los métodos personalizados, puedes invocar en cualquier momento las API proporcionadas por ofa.js para manipular elementos DOM de manera flexible y eficiente.

Relacionado con instancias

$(selector)

Función central en ofa.js, utilizada para obtener y manipular instancias de elementos DOM

 all

Obtiene todas las instancias de elementos secundarios del elemento actual

 shadow

Obtiene el shadow root del elemento actual

 prev

Obtiene el elemento hermano anterior del elemento actual

 prevs

Obtiene todos los elementos hermanos anteriores del elemento actual

 next

Obtiene el siguiente elemento hermano del elemento actual

 nexts

Obtiene todos los elementos hermanos posteriores del elemento actual

 siblings

Obtiene todos los elementos hermanos del elemento actual

 parent

Obtiene el elemento padre del elemento actual

 parents

Obtiene todos los elementos ancestros del elemento actual

 clone

Copia el elemento actual y sus elementos secundarios

 ele

Obtiene la instancia de elemento secundario del elemento actual

 root

Obtiene la instancia raíz a la que pertenece el elemento actual

 Elementos secundarios

Obtiene todos los elementos secundarios del elemento actual

 host

Obtiene el elemento host del web component al que pertenece el elemento actual

 app

Obtiene la instancia del componente o-app al que pertenece el elemento actual

Operaciones de nodos

push

Añade elementos secundarios al elemento actual

 pop

Elimina el último elemento secundario del elemento actual

 shift

Elimina el primer elemento secundario del elemento actual

 unshift

Añade el primer elemento secundario al elemento actual

 splice

Añade o elimina elementos secundarios del elemento actual

 before

Añade elementos secundarios antes del elemento actual

 after

Añade elementos secundarios después del elemento actual

 remove

Elimina el elemento actual

 wrap

Envuelve el elemento actual dentro de un elemento especificado

 unwrap

Elimina el elemento padre que envuelve al elemento actual

Operaciones de atributos

text

Obtiene o establece el contenido de texto del elemento

 html

Obtiene o establece el contenido HTML del elemento

 attr

Obtiene o establece los atributos del elemento

 css

Obtiene o establece los estilos del elemento

 style

Obtiene el objeto style del elemento

 classList

Obtiene la lista de clases del elemento

 data

Obtiene o establece los atributos data del elemento

Relacionado con eventos

on

Añade un listener de eventos al elemento

 one

Añade un listener de eventos que se ejecuta solo una vez al elemento

 emit

Dispara un evento personalizado del elemento

 off

Elimina el listener de eventos del elemento

 tag

 index

 is

Determina si el elemento coincide con un selector especificado

 refresh

 PATH

 extend

Extiende la funcionalidad de ofa.js

 version
