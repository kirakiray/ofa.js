Documentación de API

ofa.js no solo admite la sintaxis de plantillas, sino que también ofrece una rica API para manipular directamente el DOM. Al igual que jQuery, puedes usar la concisa sintaxis de ofa.js para reemplazar el verbose JavaScript nativo y realizar fácilmente operaciones en el DOM.

En las funciones del ciclo de vida y métodos personalizados de los componentes o módulos de página, puedes invocar en cualquier momento las APIs proporcionadas por ofa.js para manipular los elementos DOM de manera flexible y eficiente.

Relacionado con la instancia

$(selector)

Función principal en ofa.js para obtener y manipular instancias de elementos del DOM

 all

Obtiene todas las instancias de elementos hijos del elemento actual

 shadow

Obtiene el shadow root del elemento actual

 prev

Obtiene el elemento hermano anterior del elemento actual

 prevs

Obtiene todos los elementos hermanos anteriores del elemento actual

 next

Obtiene el siguiente elemento hermano del elemento actual

 nexts

Obtiene todos los elementos hermanos posteriores del elemento actual

 siblings

Obtiene todos los elementos hermanos del elemento actual

 parent

Obtiene el elemento padre del elemento actual

 parents

Obtiene todos los elementos ancestros del elemento actual

 clone

Clona el elemento actual y sus elementos hijos

 ele

Obtiene la instancia de elemento hijo del elemento actual

 root

Obtiene la instancia raíz a la que pertenece el elemento actual

 elementos infantiles

Obtiene todos los elementos hijos del elemento actual

 host

Obtiene el elemento host del web component al que pertenece el elemento actual

 app

Obtiene la instancia del componente o-app a la que pertenece el elemento actual

Manipulación de nodos

push

Añade un elemento hijo al elemento actual

 pop

Elimina el último elemento hijo del elemento actual

 shift

Elimina el primer elemento hijo del elemento actual

 unshift

Añade el primer elemento hijo al elemento actual

 splice

Añade o elimina elementos hijos del elemento actual

 before

Añade un elemento hijo antes del elemento actual

 after

Añade un elemento hijo después del elemento actual

 remove

Elimina el elemento actual

 wrap

Envuelve el elemento actual dentro del elemento especificado

 unwrap

Elimina el envoltorio del elemento padre del elemento actual

Manipulación de atributos

text

Obtiene o establece el contenido de texto del elemento

 html

Obtiene o establece el contenido HTML del elemento

 attr

Obtiene o establece los atributos del elemento

 css

Obtiene o establece los estilos del elemento

 style

Obtiene el objeto style del elemento

 classList

Obtiene la lista de clases del elemento

 data

Obtiene o establece los atributos data del elemento

Relacionado con eventos

on

Añade un detector de eventos al elemento

 one

Añade un detector de eventos que se ejecuta solo una vez al elemento

 emit

Dispara un evento personalizado del elemento

 off

Elimina el detector de eventos del elemento
