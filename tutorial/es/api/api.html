Documentación de API

ofa.js no solo admite la sintaxis de plantillas, sino que también ofrece una rica API para manipular directamente el DOM. Al igual que jQuery, puedes usar la concisa sintaxis de ofa.js para reemplazar el verbose JavaScript nativo y realizar fácilmente operaciones en el DOM.

En las funciones del ciclo de vida y métodos personalizados de los componentes o módulos de página, puedes invocar en cualquier momento las APIs proporcionadas por ofa.js para manipular los elementos DOM de manera flexible y eficiente.

Relacionados con instancias

$(selector)

Función central de ofa.js, utilizada para obtener y manipular instancias de elementos DOM

 all

Obtiene todas las instancias de los elementos secundarios del elemento actual

 shadow

Obtiene el shadow root del elemento actual

 prev

Obtiene el elemento hermano anterior del elemento actual

 prevs

Obtiene todos los elementos hermanos anteriores al elemento actual

 next

Obtiene el siguiente elemento hermano del elemento actual

 nexts

Obtiene todos los elementos hermanos posteriores al elemento actual

 siblings

Obtiene todos los elementos hermanos del elemento actual

 parent

Obtiene el elemento padre del elemento actual

 parents

Obtiene todos los elementos ancestros del elemento actual

 clone

Clona el elemento actual y sus elementos secundarios

 ele

Obtiene la instancia del elemento secundario del elemento actual

 root

Obtiene la instancia raíz a la que pertenece el elemento actual

 Elementos secundarios

Obtiene todos los elementos secundarios del elemento actual

 host

Obtiene el elemento anfitrión del componente web al que pertenece el elemento actual

 app

Obtiene la instancia del componente o-app al que pertenece el elemento actual

Operaciones de nodos

push

Añade un elemento secundario al elemento actual

 pop

Elimina el último elemento secundario del elemento actual

 shift

Elimina el primer elemento secundario del elemento actual

 unshift

Añade un elemento al inicio del elemento actual

 splice

Añade o elimina elementos secundarios del elemento actual

 before

Añade un elemento antes del elemento actual

 after

Añade un elemento después del elemento actual

 remove

Elimina el elemento actual

 wrap

Envuelve el elemento actual dentro de un elemento especificado

 unwrap

Elimina el envoltorio del elemento padre del elemento actual

Operaciones de atributos

text

Obtiene o establece el contenido de texto del elemento

 html

Obtiene o establece el contenido HTML del elemento

 attr

Obtiene o establece un atributo del elemento

 css

Obtiene o establece los estilos CSS del elemento

 style

Obtiene el objeto style del elemento

 classList

Obtiene la lista de clases del elemento

 data

Obtiene o establece los atributos data del elemento

Relacionados con eventos

on

Añade un escuchador de eventos al elemento

 one

Añade un escuchador de eventos que se ejecuta solo una vez al elemento

 emit

Desencadena un evento personalizado en el elemento

 off

Elimina un escuchador de eventos del elemento
