# Crear la primera aplicación

Si no quieres lidiar con otros pasos complicados y prefieres usar ofa.js para desarrollar todo el proyecto, puedes aprovechar la herramienta de construcción y vista previa en línea proporcionada oficialmente por ofa.js para crear tu primera aplicación local directamente en el navegador con un solo clic.

Esta herramienta está alojada en un servidor estático, todas las operaciones se realizan localmente en el navegador, no es necesario conectarse a un servidor dinámico, por favor, úsela con confianza.

## Iniciar el programa

Haz clic en [OFA Studio](https://core.noneos.com?redirect=studio) para abrir la herramienta de construcción ofa.js. Esta herramienta utiliza directamente el sistema de archivos local del navegador, sin necesidad de instalación ni configuración. Sigue los siguientes pasos:

1. Visita el sitio web y espera a que se cargue el programa
2. Haz clic en el botón "Crear nuevo proyecto"
3. Selecciona el tipo "Aplicación ofa.js", introduce el nombre del proyecto y luego elige el directorio local
4. En el cuadro de diálogo de confirmación de permisos, haz clic en permitir todo y luego en "Confirmar creación"
5. Tras unos segundos, el proyecto se inicializará en el directorio seleccionado

Una vez completada la inicialización del proyecto, puede usar un editor como [VSCode](https://code.visualstudio.com/) para abrir ese directorio y comenzar a escribir el código de la aplicación.
