# Crear la primera aplicación

Si no quieres complicarte con otros pasos tediosos y deseas desarrollar todo el proyecto directamente con ofa.js, puedes utilizar la herramienta de construcción y vista previa en línea proporcionada oficialmente por ofa.js para crear tu primera aplicación local con un solo clic en el navegador.

Esta herramienta está alojada en un servidor estático, todas las operaciones se realizan localmente en el navegador, sin necesidad de conectarse a un servidor dinámico, por favor, úsela con confianza.

## Iniciar el programa

Haz clic en [OFA Studio](https://core.noneos.com?redirect=studio) para abrir la herramienta de construcción de ofa.js. Esta herramienta utiliza directamente el sistema de archivos local del navegador, sin necesidad de instalación ni configuración. Sigue los pasos a continuación:

1. Visita el sitio web y espera a que el programa cargue
2. Haz clic en el botón "Crear nuevo proyecto"
3. Selecciona el tipo "aplicación ofa.js", ingresa el nombre del proyecto y luego elige el directorio local
4. En el cuadro de diálogo de confirmación de permisos que aparece, permite todo y luego haz clic en "Confirmar creación"
5. Después de unos segundos, el proyecto se inicializará en el directorio seleccionado

Después de completar la inicialización del proyecto, puede usar editores como [VSCode](https://code.visualstudio.com/) para abrir el directorio y comenzar a escribir el código de la aplicación.
