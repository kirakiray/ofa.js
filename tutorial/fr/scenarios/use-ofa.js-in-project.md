# Utiliser les prompts IA d'ofa.js dans un projet

Étant donné qu'ofa.js n'est pas encore devenu un framework largement reconnu, les principaux modèles d'IA ne possèdent pas encore la capacité d'utiliser directement ofa.js. Pour cette raison, nous avons soigneusement préparé des prompts dédiés pour aider l'IA à apprendre et à consulter les méthodes d'utilisation d'ofa.js.

Nous proposons deux versions de l'invite :

## Version simplifiée des prompts

Ceci est une version concentrée et optimisée, conçue pour minimiser la consommation de jetons d’entrée préliminaires et adaptée à la plupart des scénarios :

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/tiny/start.md
```

En utilisant cette invite, l’IA peut développer efficacement des composants ou des modules de page pour ofa.js.

## Prompt complet

Si l'IA que vous utilisez n'est pas assez intelligente, vous pouvez essayer d'utiliser la version complète non condensée de l'invite. Bien qu'elle occupe plus de tokens lors de l'initialisation, elle peut offrir de meilleurs résultats :

```
https://raw.githubusercontent.com/ofajs/ofa.js/main/llms/origin/start.md
```

En fournissant ces mots-clés, nous espérons aider les développeurs à utiliser plus facilement les outils d'IA pour le développement du projet ofa.js, améliorant ainsi l'efficacité du développement.
