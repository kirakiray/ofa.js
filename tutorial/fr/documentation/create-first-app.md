# Créer la première application

Si vous ne voulez pas vous embêter avec d’autres étapes fastidieuses et souhaitez développer l’ensemble de votre projet directement avec ofa.js, vous pouvez utiliser l’outil de construction et de prévisualisation en ligne officiel d’ofa.js pour créer en un clic votre première application locale dans le navigateur.

Cet outil est hébergé sur un serveur statique ; toutes les opérations sont effectuées localement dans le navigateur, sans connexion à un serveur dynamique. Utilisez-le en toute confiance.

## Lancement du programme

Cliquez sur [OFA Studio](https://core.noneos.com?redirect=studio) pour ouvrir l'outil de construction ofa.js. Cet outil utilise directement le système de fichiers local du navigateur, sans nécessiter d'installation ou de configuration. Suivez les étapes suivantes :

1. Visitez le site web et attendez le chargement du programme
2. Cliquez sur le bouton "Créer un nouveau projet"
3. Sélectionnez le type "application ofa.js", saisissez le nom du projet, puis choisissez un répertoire local
4. Dans la boîte de dialogue de confirmation des autorisations qui apparaît, cliquez sur "Autoriser" pour toutes les options, puis cliquez sur "Confirmer la création"
5. Après quelques secondes, le projet sera initialisé dans le répertoire sélectionné

Après l'initialisation du projet, vous pouvez utiliser un éditeur tel que [VSCode](https://code.visualstudio.com/) pour ouvrir ce répertoire et commencer à écrire le code de l'application.
