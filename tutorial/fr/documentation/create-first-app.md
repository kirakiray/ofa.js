# Créer votre première application

Si vous ne voulez pas vous embêter avec d’autres étapes fastidieuses et souhaitez directement développer l’ensemble de votre projet avec ofa.js, vous pouvez utiliser l’outil de construction et de prévisualisation en ligne officiel d’ofa.js pour créer en un clic votre première application locale dans le navigateur.

Cet outil est hébergé sur un serveur statique, toutes les opérations sont effectuées localement dans le navigateur, sans nécessiter de connexion à un serveur dynamique, vous pouvez l'utiliser en toute confiance.

## Démarrer le programme

Cliquez sur [OFA Studio](https://core.noneos.com?redirect=studio) pour ouvrir l'outil de construction ofa.js. Cet outil utilise directement le système de fichiers local du navigateur, sans nécessiter d'installation ou de configuration. Suivez les étapes ci-dessous :

1. Accédez au site Web et attendez que le programme se charge
2. Cliquez sur le bouton « Créer un nouveau projet »
3. Sélectionnez le type « Application ofa.js », saisissez le nom du projet, puis choisissez le répertoire local
4. Dans la boîte de dialogue de confirmation des autorisations, cliquez sur « Autoriser » pour tout, puis sur « Confirmer la création »
5. Après quelques secondes, le projet sera initialisé dans le répertoire choisi

Une fois l'initialisation du projet terminée, vous pouvez utiliser [VSCode](https://code.visualstudio.com/) et d'autres éditeurs pour ouvrir ce répertoire et commencer à écrire le code de l'application.
