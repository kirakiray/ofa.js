# Introduction

## Qu'est-ce que ofa.js ?

ofa.js est un framework Web front-end très facile à utiliser, conçu pour construire des applications Web, visant à améliorer l'efficacité de développement et à réduire le seuil d'entrée.

> Si vous êtes **déjà familier** avec HTML, CSS et JavaScript, alors ofa.js est un excellent choix.

## Pourquoi ai-je développé ofa.js ?

Bien que des frameworks frontaux tels que React et Vue existent déjà, ils présentent tous un seuil d’apprentissage relativement élevé. Les développeurs doivent d’abord maîtriser une série de connaissances préalables, comme l’installation de Node.js, l’utilisation de npm pour installer des paquets, la mise en place d’un environnement de développement, etc.

Ces étapes initiales ne sont pas indispensables pour le développement réel, mais sont devenues une "boîte noire" au stade de lancement du projet, faisant porter à l'application une complexité inutile dès le départ. La naissance d'ofa.js vise précisément à briser cette impasse.

## But de ofa.js

### Simplification du processus de développement front-end

ofa.js simplifie les processus fastidieux du développement front-end traditionnel. Les développeurs peuvent rapidement créer des applications front-end sans avoir à apprendre des outils comme Node.js, npm et les échafaudages. Il suffit d'ouvrir le programme de construction fourni par l'officiel (version web) via un navigateur, puis de sélectionner un répertoire local pour commencer le développement.

### Réduire le seuil d'entrée au développement

Beaucoup souhaitent créer leur propre application ou devenir développeur, mais, confrontés à la voie traditionnelle du développement Web (y compris Android, iOS, etc.), ils renoncent dès le départ, intimidés par la masse de connaissances préalables requises : installer l’environnement de développement, configurer le boilerplate, entre autres.

ofa.js s'engage à simplifier tout. Il suffit d'un navigateur Chrome, ouvrez une page web pour construire des applications, sélectionnez et ouvrez directement un répertoire local, éliminant les préparatifs avant le développement. Tous les calculs, opérations de données et de stockage sont exécutés localement sur l'appareil de l'utilisateur, sans dépendre de services cloud.

### Devenir la solution ultime des micro-frontends

ofa.js permet au développement front-end Web d’être segmenté en plusieurs modules indépendants, comme dans une architecture micro-services : chaque module peut être développé et déployé séparément. Une fois les frontières du front-end Web traditionnel abolies, les technologies front-end dépasseront progressivement les limites des technologies serveur.

Les microservices (Microservices) sont un style d'architecture logicielle qui décompose des applications volumineuses et complexes en plusieurs petits services à granularité fine, déployés et exécutés indépendamment.
