API 文檔

ofa.js 不僅支持模闆語法，還提供豐富的 API 來直接操作 DOM。就像 jQuery 一樣，妳可以用簡潔的 ofa.js 語法替代冗長的原生 JavaScript，輕鬆完成 DOM 操作。

在組件或頁面模塊的生命周期函數及自定義方法中，妳都可以隨時調用 ofa.js 提供的 API，靈活高效地操控 DOM 元素。

實例相關

$(selector)

ofa.js 中的覈心函數，用於獲取和操作 DOM 元素實例

 all

獲取當前元素的所有子元素實例

 shadow

獲取當前元素的 shadow root

 prev

獲取當前元素的前一個兄弟元素

 prevs

獲取當前元素之前的所有兄弟元素

 next

獲取當前元素的下一個兄弟元素

 nexts

獲取當前元素之後的所有兄弟元素

 siblings

獲取當前元素的所有兄弟元素

 parent

獲取當前元素的父元素

 parents

獲取當前元素的所有祖先元素

 clone

復製當前元素及其子元素

 ele

獲取當前元素的子元素實例

 root

獲取當前元素所屬的根實例

 子元素

獲取當前元素的所有子元素

 host

獲取當前元素所屬的 web component 宿主元素

 app

獲取當前元素所屬的 o-app 組件實例

節點操作

push

添加當前元素的子元素

 pop

刪除當前元素的最後一個子元素

 shift

刪除當前元素的第一個子元素

 unshift

添加當前元素的第一個子元素

 splice

添加或刪除當前元素的子元素

 before

在當前元素之前添加子元素

 after

在當前元素之後添加子元素

 remove

移除當前元素

 wrap

將當前元素包裹在指定元素中

 unwrap

移除當前元素的父元素包裹

屬性操作

text

獲取或設置元素的文本內容

 html

獲取或設置元素的 HTML 內容

 attr

獲取或設置元素的屬性

 css

獲取或設置元素的樣式

 style

獲取元素的 style 對象

 classList

獲取元素的類名列錶

 data

獲取或設置元素的 data 屬性

事件相關

on

爲元素添加事件監聽器

 one

爲元素添加隻執行一次的事件監聽器

 emit

觸發元素的自定義事件

 off

移除元素的事件監聽器

其他

o-app組件

ofa.js 應用的根組件，用於管理應用狀態和路由

 o-page組件

ofa.js 的頁面組件，用於創建單頁應用頁面

 formData

獲取錶單數據

 tag

獲取元素的標籤名

 index

獲取元素在其父元素中的索引

 is

判斷元素是否匹配指定選擇器

 refresh

刷新當前組件或頁面

 PATH

獲取當前頁面的路徑

 extend

擴展 ofa.js 的功能

 version

獲取 ofa.js 的版本號
