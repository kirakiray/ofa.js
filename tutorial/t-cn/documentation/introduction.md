# 介紹



## 什麼是 ofa.js？



ofa.js 是一款非常易用的 Web 前端框架，用於構建 Web 應用，旨在提高開發效率，降低開發門檻。

>如菓妳對 HTML、CSS 和 JavaScript **已經基本熟悉**，那麼 ofa.js 是一個非常好的選擇。

## 爲什麼我要開發 ofa.js？



雖然已有 React、Vue 等前端框架，但牠們都存在較高的上手門檻。開發者需要先學習一系列前置知識，例如安裝 Node.js、使用 npm 安裝包、搭建腳手架等。

這些初始步驟對實際開發並非不可或缺，反而成爲瞭項目啓動階段的「黒箱」，讓應用從一開始就揹負瞭不必要的復雜度。ofa.js 的誕生，正是爲瞭打破這一睏境。

## ofa.js 存在的目的



### 簡化前端開發流程



ofa.js 簡化瞭傳統前端開發的繁瑣流程。開發者無需學習 Node.js、npm 和腳手架等工具，卽可快速搭建前端應用。隻需通過瀏覽器打開官方提供的構建程序（網頁版），然後選擇本地目錄，便可開始開發。

### 降低開發門檻



許多人希望開發屬於自己的應用，或成爲一名開發者，但面對傳統 Web（包括安卓、iOS 等）開發路綫時，卻因需要學習大量前置知識，例如安裝開發環境、配置腳手架，這讓許多初學者知難而退。

ofa.js 緻力於讓一切變得簡單。隻需一個 Chrome 瀏覽器，打開網頁卽可構建應用，直接選擇並打開本地目錄，省去開發前的準備工作。所有計算、數據和存儲操作均在用戶本地執行，無需依賴雲端服務。

### 成爲微前端的終極解決方案



ofa.js 讓 Web 前端開發可以像微服務一樣，細分爲多個獨立模塊，每個模塊均可獨立開發、獨立部署。當傳統 Web 前端的邊界被打破後，前端技術將逐步突破服務器技術的邊界。

> 微服務（Microservices）是一種軟件架構風格，牠將大型、復雜的應用拆分爲多個細粒度、獨立部署和運行的小型服務。
