# 創建第一個應用



如菓妳不想摺騰其他繁瑣步驟，想直接使用 ofa.js 開發整個項目，可以藉助 ofa.js 官方提供的在綫構建與預覽工具，在瀏覽器裏一鍵創建妳的第一個本地應用。

該工具託管於靜態服務器，所有操作均在瀏覽器本地完成，無需連接動態服務器，請放心使用。

## 啓動程序



點擊 [OFA Studio](https://core.noneos.com?redirect=studio) 打開 ofa.js 構建工具。該工具直接使用瀏覽器的本地文件系統，無需安裝或配置。按照以下步驟操作：

1. 訪問網站並等待程序加載
2. 點擊 "創建新項目" 按鈕
3. 選擇 "ofa.js 應用" 類型，輸入項目名稱，然後選擇本地目錄
4. 在彈齣的權限確認對話框中全部點擊允許，然後點擊 "確認創建"
5. 等待幾秒後，項目將在所選目錄中初始化完成

項目初始化完成後，您可使用 [VSCode](https://code.visualstudio.com/) 等編輯器打開該目錄，開始編寫應用代碼。
